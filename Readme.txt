Django Inventory Mangement System Assignment.

This project is having functionality of :
1.	Adding, Editing Products Id
2.	Adding, Editing Location Id
3.	Adding, Editing Product Movement
4.	From Location will set to blank when moving product to a location
5.	To Location will set to blank when moving out product
6.	Report page will show all the updated information of Location, Product and Quantity
